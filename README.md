# spring-runoff-modelling-iska
# 🌊 ГИС-моделирование весеннего стока р. Иска (Бассейн р. Тобол)
**Разработка базы данных и модуля для автоматизированного гидрологического моделирования**

## 📖 О проекте
Данный репозиторий содержит результаты выпускной квалификационной работы (бакалавриат) по направлению "Картография и геоинформатика".

**Цель работы**: создание геопространственной базы данных и QGIS-плагина для моделирования формирования весеннего водного стока на основе ландшафтно-гидрологического подхода Калинина В.М. для бассейна р. Иска (Западная Сибирь).

**Ключевые особенности проекта:**
*   🗺️ **Оцифровка**: Выделение 1176 стокоформирующих комплексов (СФК), сгруппированных в 24 типа ландшафтов.
*   🐘 **Хранение данных**: Реляционная база геоданных на базе **PostgreSQL + PostGIS**, содержащая гидрологические параметры для каждого комплекса.
*   🧮 **Автоматизация**: Модуль для **QGIS** (Python/PyQt), полностью автоматизирующий расчет весеннего и годового стока по формулам В.М. Калинина.
*   🤖 **ИИ-ассистент**: При разработке модуля использовался искусственный интеллект (DeepSeek) для кодогенерации и оптимизации логики вычислений.
